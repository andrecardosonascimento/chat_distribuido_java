# CHAT DISTRIBUIDO EM JAVA

Projeto simples de um **CHAT EM REDE**, desenvolvido como demostração prática de funcionamentos de **SISTEMAS DISTRIBUIDOS**, As principais funcionalidades do Projeto Comunicação Cliente-Servidor via TCP/IP, Servidor Que aceita multimas conexões simultâneas, Mensagens são enviadas a todos os usários conectados, Usando interface via terminal (CLI).

## Tecnologias
- Java 17+
- Sockets (java.net)
- Threads
- ConcurrentHashMap para controle seguro de clientes
